# 分子可视化工具

## 网站简介
本项目是一个基于Web的分子可视化工具，旨在为用户提供一个直观、便捷的方式来观察和交互分子结构。该工具利用先进的3D可视化技术和化学信息学算法，使用户能够轻松加载、查看和操作分子模型。

## 页面与功能
网站包含8个主要页面，分别提供以下功能：
1. 首页：介绍分子可视化工具的基本功能和特点。
2. 分子加载：支持用户上传或输入分子结构数据（如PDB、SDF格式），并在3D视图中显示。
3. 分子查看：提供多种视角和显示模式（例如：球棍模型、空间填充模型等），以适应不同的观察需求。
4. 分子操作：允许用户对分子进行旋转、缩放、平移等交互操作。
5. 属性分析：展示分子的基本属性，如分子量、分子式等。
6. 化学反应模拟：支持简单的化学反应模拟，帮助用户理解分子间的相互作用。
7. 帮助文档：提供详细的使用指南和技术支持信息。
8. 关于我们：介绍项目背景、开发团队和联系方式。
